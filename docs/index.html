CYD를 USB로 연결하고 버튼을 누르면 끝. 앱도, 개발 환경도 필요 없습니다.

이 브라우저는 Web Serial을 지원하지 않습니다 — Chrome 또는 Edge로 열어 주세요. 이 페이지는 HTTPS 또는 localhost에서만 동작합니다.

Features

NTP로 한국 표준시를 자동 동기화하고, 이후에도 스스로 재동기화합니다.

알아서 어두워지는 화면

내장 조도센서가 방 밝기를 감지해 밤에는 은은하게, 낮에는 선명하게.

터치 한 번으로 조작

왼쪽 탭 12/24시간제 · 오른쪽 탭 밝기 모드 · 5초 누름 WiFi 초기화

Guide

설치는 3분이면 끝

1
USB로 연결

데이터 전송이 되는 케이블로 PC와 연결하세요. 충전 전용 케이블은 인식되지 않습니다.

2
기기에 설치하기

위의 버튼을 누르고 목록에서 CH340 포트를 선택하면 자동으로 설치됩니다.

3
휴대폰으로 WiFi 연결

기기 화면 안내에 따라 휴대폰 WiFi 목록에서 CYD-Clock에 접속하고, 자동으로 뜨는 창에서 집 WiFi를 선택하세요. 이후에는 전원만 꽂으면 됩니다.

Help

문제가 있나요?

포트 선택 창에 아무것도 안 나와요
CH340 드라이버가 없는 경우입니다. 위의 드라이버 링크에서 설치한 뒤 케이블을 뽑았다 다시 꽂아 주세요. 충전 전용 케이블도 인식되지 않는 원인입니다.
설치 중 "Failed to initialize" 오류가 나요
다른 프로그램(시리얼 모니터, Arduino IDE 등)이 포트를 사용 중이면 실패합니다. 모두 닫고 다시 시도하세요. 일부 보드는 설치 시작 시 BOOT 버튼을 누른 채 연결해야 할 수 있습니다.
화면 색이 반전되어 보여요 (배경이 흰색)
일부 CYD 변종 패널의 특성입니다. src/main.cpp의 PANEL_INVERT를 true로 바꿔 다시 빌드하면 해결됩니다.
시간이 "--:--"로만 나와요
WiFi는 연결됐지만 NTP 동기화가 아직 안 된 상태입니다. 잠시 기다리거나 공유기의 인터넷 연결을 확인하세요. 시계 화면을 5초간 길게 누르면 WiFi를 다시 설정할 수 있습니다.
